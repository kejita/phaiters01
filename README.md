# Pureact(ピュアクト) - みんなをつなぐ温かなコミュニティ

## プロジェクト概要

「Pureact(ピュアクト)」は、がん患者さんとその家族、医療従事者を温かくつなぐコミュニティアプリの紹介Webサイトです。
「温もり」「清潔感」「透明性」をコンセプトに、視認性が高く操作に迷わないシンプルな構成を実現しました。

## 🎯 ターゲットユーザー

- がん患者本人
- 患者の家族
- 医療従事者
- サポーター

## 💎 コアバリュー

- **繋がりによる心のケア**: 同じ経験をした人々とのつながり
- **正しい情報の共有**: 医療専門家監修の信頼できる情報
- **安全なコミュニティ**: 匿名性確保とプライバシー保護

## 🌟 主な機能

### 1. チャット機能
- 匿名で安心して相談できるチャット機能
- 同じ経験をした人からの温かい励まし

### 2. 掲示板
- 様々なテーマ別の掲示板
- 経験や情報を共有できる場所

### 3. イベント情報
- NPO法人が運営するイベントや勉強会の情報
- リアルなつながりを大切に

### 4. プライバシー保護
- 個人情報の厳重な保護
- 匿名性の確保

## 🎨 デザインコンセプト

### カラーパレット
- **プライマリカラー**: #E8A798（温かいピンク系）
- **セカンダリカラー**: #A8C8E8（清潔感のあるブルー）
- **アクセントカラー**: #F4A261（温かみのあるオレンジ）

### デザイン原則
- **温もり**: 暖かい色合いと柔らかい影
- **清潔感**: シンプルで整理されたレイアウト
- **透明性**: 明確な情報階層と直感的なナビゲーション

## 🔧 セットアップ方法

1. リポジトリをクローンまたはダウンロード
2. ファイルをWebサーバーにアップロード
3. ブラウザで `index.html` にアクセス

### ローカル開発環境
```bash
# Pythonを使用した簡易サーバー
python -m http.server 8000

# Node.jsを使用した場合
npx serve .
```
